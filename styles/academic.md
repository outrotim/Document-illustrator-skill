## 学术汇报风格 PPT

### 概述
专业、简洁、信息密集的学术演示风格，适合科研汇报、论文答辩、学术会议等场景。

### 提示词

Create a professional academic presentation slide with HIGH INFORMATION DENSITY, suitable for scientific research presentations.

Visual Style Requirements:
- Clean white or light gray background
- Professional color scheme: navy blue (#1a365d), teal (#0d9488), dark gray (#374151)
- Accent colors: burgundy (#7f1d1d) or forest green (#166534) for highlights
- Modern sans-serif typography

HIGH DENSITY Layout Guidelines:
- Multiple content sections with clear visual separation
- Use bullet points, numbered lists, and key-value pairs
- Include data visualization elements: mini charts, percentage indicators, key metrics
- Divide slide into 2-3 columns or grid layout when appropriate
- Show specific numbers, statistics, and quantitative data
- Include methodology flowcharts or process diagrams
- Add icons and symbols to represent concepts visually

Design Elements:
- Scientific diagrams: molecular structures, pathway diagrams, network graphs
- Data cards with key findings and statistics
- Timeline or flowchart elements for processes
- Comparison tables or matrices
- Mini infographics embedded in content

Typography (Chinese friendly):
- Large bold title in Chinese
- Clear hierarchical subtitles
- Compact but readable body text
- Key terms and numbers highlighted

Overall Requirements:
- Information-rich but organized
- Professional academic aesthetic
- Suitable for peer review audiences
- Balance between text and visual elements
- Show actual data points and findings
